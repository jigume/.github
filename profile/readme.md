## 프로젝트 주요 기능 (MVP)

### 지도 기반 공동 구매처 찾기
지구미에서 공동 구매를 이용하는 사용자들은 직접 픽업을 받으러 가요. 
"카카오 지도 API"를 활용한 서비스로 내 위치 기반의 공동 구매를 열거나 찾아갈 수 있도록 했어요.
